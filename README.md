## ğŸš€ Getting Started

### Local Development

1. **Clone or download the repository:**
   ```bash
   git clone https://github.com/parveenmou/Portfolio-vs.git
   cd Portfolio-vs
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start local development server:**
   ```bash
   npm start
   ```
   
   The portfolio will open at `http://127.0.0.1:8080`
